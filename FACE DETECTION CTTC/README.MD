# Facial Recognition System

## 📌 Description
This project detects and recognizes human faces using computer vision.

## ⚙️ Technologies Used
- Python
- OpenCV
- NumPy

## 🚀 Features
- Face detection
- Face recognition
- Real-time camera support

## ▶️ How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run the program:
   python main.py

## 📂 Dataset
Dataset is not uploaded due to large size.
